下拉框指令说明：

lar-select
下拉框数据,默认数据格式： [ { "id": "document", "name": "文档资源", "data":"" }, { "id": "picture", "name": "图片资源", "data":"" } ]
lar-option
提供额外选择项(如：全部、请选择、不限等)
lar-multi
是否多选，默认单选
lar-search-able
是否显示搜索框，默认不显示
lar-model
要绑定的数据(字符串，数组)都支持 ,说明：字符串以默认以逗号隔开，如('1003,1002,1001')
