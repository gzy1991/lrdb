侧边栏Slidebar

样例代码

require(["PDUtilDir/slidebar"], function(Slidebar){
      var slidebar = Slidebar({
          url : getServer() + "/static/core/examples/util/slidebar/views/template.html",
          afterLoad : function($Panel) {
              console.log("模板加载完成，可以使用了");
          }
      });
})

配置属性说明

名称 类型 默认值 描述
sid string 自动生成 组件唯一id，用来设置组件dom的id
id string 页面dom节点id，用来初始化侧边栏内容
body string 侧边栏内容
url string 远程页面的url，用来初始化侧边栏内容
urlCache boolean true 是否缓存远程url对应的模板
width float 侧边栏宽度[不支持页面自适应]
size string 侧边栏宽度[可选值有"lg"和"sm"][支持页面自适应]
cache boolean false 是否缓存组件。缓存后sid做为缓存key，再次初始化时通过sid查找相应缓存的组件 ，
afterLoad function null 侧边栏内容设置/加载完成回调
afterOpen function null 侧边栏打开完成回调
afterClose function null 侧边栏关闭完成回调
onDestroy function null 侧边栏销毁时回调

对象方法说明

名称 类型 参数 描述
close function 无 关闭侧边栏
open function 无 打开侧边栏
destroy function 无 销毁侧边栏
setBody function html 设置侧边栏html
