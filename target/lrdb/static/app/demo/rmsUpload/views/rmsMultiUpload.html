参数及方法说明

组件说明
  • 支持上传文件的编辑功能
  • 支持已上传数据初始化
  • 支持编辑内容自定义配置
注：各类文件支持编辑的格式如下，除此之外均不支持编辑。文件大小均精确到小数点后两位，除以下格式文件外，单位均为Byte：
doc : 'doc docx ppt pptx xls xlsx pdf';文档大小的单位为KB
img : 'png jpg bmp tiff gif';图片大小的单位为KB
audio : 'mp3 wma wav';音频大小的单位为MB
video : 'mp4 flv avi wmv rmvb mov mpg';视频大小的单位为GB
方法
  • multiUploader.getAllData($scope,uploader) -   调用以获取对应上传附件的数据，数据会绑定到$scope[uploader]上。uploader是上传组件实例化对象名称。
  • multiUploader.init($scope,config) -   初始化多个上传组件，config为参数。
参数
  • uploader - String  组件实例化对象的唯一名称，不可重复，默认'multi-uploader'
  • edit - Boolean  是否可编辑，true为可编辑，默认为false。
  • param - Object  设置上传组件模式
    • placeAt - String  组件容器的DOM节点选择器，默认'#multi'
    • file - String  主要用于控制上传组件中编辑模板的选择，指定一个模板后可避免重复发请求获取html文件；次要用于计算文件大小。默认为''，即所有类别文档将根据文件后缀进行判断从而选择编辑模板
  • data - Object|Boolean  初始化数据，即已上传附件数据，默认为false。 
    object-data的属性名称必须与settings-uploader中key的对应字段相符。注：初始化inputSelect的数据必须使用数组
