字典管理后台Service示例

queryFromRootByCode

url：/sword/sys/dict/queryFromRootByCode

resumes：该方法只针对ParentID为ROOT的根节点，根据code查询节点及其一级子节点（或者整棵树）的列表

parameters：String code 传入的code，根据code查询指定节点和其下子节点，为空则返回查询失败
boolean queryAllTree 可不传，内置默认为false，查询一级子节点，若为true，则查询整棵树

remarks: 该方法只针对simple字典，即只有一层叶子节点的树状字典。

queryDictTreeByRootCode

url：/sword/sys/dict/queryDictTreeByRootCode

resumes：根据rootCode查询一棵树，无论纵深。

parameters：String rootCode 标记一颗树的专用字段，值为一棵树根节点的id的值。如果为空，则查询字典表全部数据

querySubDictList

url：/sword/sys/dict/querySubDictList

resumes：根据parentId查询下一级子节点。

parameters：String parentId 父节点的id，为空时查询所有根节点

queryCurAndSubDictList

url：/sword/sys/dict/queryCurAndSubDictList

resumes：根据传入uuid，查询id或parentId等于uuid的所有字典项列表，即返回当前节点及其一级子节点。

parameters：String uuid 传入的id或parentId，为空时返回查询失败

queryDictItemById

url：/sword/sys/dict/queryDictItemById

resumes：根据传入id，查询对应的节点。

parameters：String id 传入的id，为空时返回查询失败

saveOrUpdateDictItem

url：/sword/sys/dict/saveOrUpdateDictItem

resumes：保存或更新字典项。

parameters：PdWebDictionary dictItem 传入的对象判断保存或更新

deleteDictItems

url：/sword/sys/dict/deleteDictItems

resumes：删除指定父节点下选中的字典项。

parameters：String parentId 待删除节点的父节点id
String ids 需要删除的字典项的id序列，以逗号分隔

validateDictItem

url：/sword/sys/dict/validateDictItem

resumes：验证指定节点的code或name在同一父节点下是否重复。验证通过则返回true

parameters：String parentId 待验证节点的父节点id
String id 待验证节点的id
String dictName/dictCode 待验证节点的name或code

validateDictClassification

url：/sword/sys/dict/validateDictClassification

resumes：验证根节点的classification在同级节点中是否重复。验证通过则返回true

parameters：String dictClassification 待验证节点的业务分类编码

queryByClassifications

url：/sword/sys/dict/queryByClassifications

resumes：根据多个树的分类名称查询多棵树，以map的形式返回

parameters：String classifications 多棵树的业务分类编码，以英文逗号分隔

queryByCode*

url：/sword/sys/dict/queryByCode

resumes：根据某个树的分类名称查询指定code的字典项

parameters：
String dictClassification 指定树的业务分类编码
String selfCode 指定节点的code
boolean queryChildren 是否查询子节点，不传则默认false

queryByCodes

url：/sword/sys/dict/queryByCodes

resumes：根据某个树的分类名称查询指定一组code对应的多个字典项，以Map返回

parameters：
String dictClassification 指定树的业务分类编码
String codeArray 指定的多个节点的code，以英文逗号分隔

queryByCodeAndParent*

url：/sword/sys/dict/queryByCodeAndParent

resumes：根据某个树的分类名称查询指定父code下的指定code对应的字典项

parameters：
String dictClassification 指定树的业务分类编码
String parentCode 指定节点的父节点code
String selfCode 指定节点的code
boolean queryChildren 是否查询子节点，不传则默认false

queryByCodeAndRemarks*

url：/sword/sys/dict/queryByCodeAndRemarks

resumes：根据某个树的分类名称查询指定remarks的指定code对应的字典项

parameters：
String dictClassification 指定树的业务分类编码
String remarks 指定的remarks值
String selfCode 指定节点的code
boolean queryChildren 是否查询子节点，不传则默认false

注意：

标识*的Service所查询的指定classification内的code的唯一性由业务保证， 若这些service查询返回结果不唯一，则默认取第一个匹配的code返回。
