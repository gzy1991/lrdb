功能简介

根据输入内容自动匹配搜索结果,依赖jQuery,bootstrap等

API

公共函数

Typeahead(config)
初始化函数,config请参考下方参数列表

参数列表

  • id(必填) - 类型:String
    定位页面元素锚点, 初始化组件的唯一标示
  • btn - 类型:String
    不设置该属性, 则不启用按钮
    设置该属性, 则启用按钮, 属性值为按钮名称
  • data - 类型:String|Array
    设置组件数据源,有两种类型:
    • String - 远程数据源地址
    • Object - 前端数据源数组
  • lazyMatch - 类型:Boolean, 默认值:true
    是否启用延迟匹配
  • key - 类型:Object (历史遗留问题,适当时候调整api)
    设置字段转换map对象, 属性如下
    • id - 数据ID字段
    • data - 数据文本字段
  • dataFormat - 类型:Array (历史遗留问题,适当时候调整api)
    数据字段对象数组, 设置每个字段的输出格式, 属性如下
    • name - 标题名
    • field - 字段名
    • width - 内容宽度
  • callback - 类型:Function
    选中数据后的处理函数
  • panelCss - 类型:Object (不推荐使用)
    一种怪异的对于内容显示不全的解决方案, 属性如下
    • width - 类型:Number, 数据面板宽度, 相对于输入框宽度的倍数
    • align - 类型:String, 数据面板对齐方向, left|right
